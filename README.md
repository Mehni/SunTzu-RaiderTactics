# SunTzu-RaiderTactics

Version: 0.17.0.1

# Description
Adds a new and challenging raid type. Pirate raiders will split up to try and flank you.

# How to use
Step 1. Install mod. Save to add or remove from any existing savegame.
Step 2. Get rekt.
